✦ study dashboard
A tiny desktop widget that lives on your Mac and gives you instant access to everything you need to study — no searching, no distractions, just one click.
Built with Electron. Launches automatically every time you open your laptop.

what it does
Floating softly on your screen, study hub gives you four instant shortcuts:

📧 Gmail — check your inbox
💻 LeetCode — jump straight to your profile
🐙 GitHub — open your dashboard
🎧 White Noise — your focus playlist, one click away

Click any icon → your browser opens it instantly.

setup
bashgit clone https://github.com/Shreywsi/study-dashboard.git
cd study-dashboard
npm install
npm start
That's it. Do this once and it launches automatically every time you start your Mac — no VS Code, no terminal, nothing.

built with

Electron — desktop app framework
Vanilla HTML, CSS & JS
